Abstract

This is a proposed set of mappings between WAI-ARIA Math Module features and platform accessibility APIs to support accessibility of Math content.

Status of This Document

This specification was published by the Getting Math onto Web Pages Community Group. It is not a W3C Standard nor is it on the W3C Standards Track.
